# Quiz Game Trivia (Python)
A Python console-based quiz game built using Object-Oriented Programming and modular design. The program loads question data (using the Open Trivia Database), creates question objects, and manages quiz flow, user input, and scoring through separate classes and modules.

# ▶️ How to Run

1. Make sure you have Python installed on your computer.

2. Download or clone this repository.

3. Ensure all files are in the same folder:

   main.py
   question_model.py
   quiz_brain.py
   data.py

4. Open a terminal (or your preferred Python IDE).

5. Navigate to the folder containing the files.

6. Run the script using:

   python main.py

7. Follow the on-screen prompts to answer the quiz questions.
